## music163.py

下载网易云音乐 mp3 + 歌词，下载的 mp3 文件有 tag 信息
>
    1. 修改源文件的 url (网易云音乐歌单地址) 和 output (保存路径)
    2. 运行 music163.py
>
    测试环境：macOS + Python3 + 100块钱以内的mp3/mp4

    需要的库有点儿多，提示需要什么就安装什么吧。
    libmagic 问题参照下面的链接解决依赖问题。
